Compilation
-----------
1. clone directory
    '''
    https://github.com/sedgwickc/RoverDaemons.git
    '''
2. navigate to newly created RoverDaemons directory
3. run make
    '''
    make all
    '''

Startup 
-------

1. create channels 
    '''
    sudo ./create_channels.sh -c
    '''
2. start daemons
    '''
    sudo ./build/sub_nav
    '''
3. start rover controller
    '''
    sudo ./build/control
    '''

Troubleshooting
---------------
1. delete channels
    '''
    sudo ./create_channels.sh -d
    '''
